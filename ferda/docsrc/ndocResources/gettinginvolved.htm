Contribute to the Success of Ferda DataMiner

There are many ways you can become involved in the Ferda DataMiner development effort.

  • Participating in the ferda-users email list is the easiest way to share your tips and tricks for using Ferda DataMiner with other users.
  • If you find bugs or have suggestions for new feature please use the Ferda DataMiner tracker database so that we can keep current with our user's needs.
  • If you have some time to spare, become part of the Ferda DataMiner developer team. There are plenty of new features we are working on implementing, and there are always bugs to fix, and the more people we have working on Ferda DataMiner, the better it becomes. To join the effort, contact one of the project admins via the Ferda DataMiner SourceForge foundry, and we will get you set up.

Thanks for using Ferda DataMiner.
