Ferda Data Miner

Česky

Ferda is an application for datamining. Currently it is an application which replaces the old user interface for the datamining system LISp-Miner by the new one, which is more eye-candy and user-friendly. Ferda brings in the view of the datamining task as a record of a function using visual objects - boxes. The application is of modular design to make its improvement easy and it is prepared for the distribution of its modules and computations through network using Ice. In future an implementation of ontology and expert systems is going to be added to Ferda.

MFF (Faculty of Mathematics and Physics) project

Ferda originated as a software project at Faculty of Mathematics and Physics of Charles university in Prague.
