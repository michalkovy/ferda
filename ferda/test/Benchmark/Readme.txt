See http://www.yoda.arachsys.com/csharp/benchmark.html for the info about benchmark framework used.

Execution
Benchmark.exe [options]
or
Benchmark.exe [options] <iterations>

Options are:
-version	Prints information about system
-runtwice   Executes benchmarks twice (the second run could be different thanks to jitter)

<iterations> is number of iterations used, default is 10000

Please read and fill Results.txt file
